How To run uvspec

  • Radiative transfer problems may be hard. Please make sure you have a sufficient understanding of the of the problem you want to solve. Read a good textbook on radiative transfer theory before starting uvspec.
  • A quick overview of how to run uvspec is available on-line: uvspec basic usage.
  • A complete description of all variables are provided in the libRadtran User's Guide.
  • To run uvspec do as follows:
    • Specify input variables.
    • Save the input file. (Only bold face variables are written to file)
    • Push Run to execute uvspec.
    • Look at output with the Plot facility.
    • Save output file.
  • Examples of uvspec input and output files are provided with the libRadtran distribution. Please check the examples directory.
  • If you run into problems, please check the libRadtran User's Guide and the libRadtran FAQ.
